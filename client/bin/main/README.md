# Emulator Module (`client`)

This folder contains the emulator/server-side logic for the project.

## Build

From repository root:

- `./gradlew :emulator:compileJava`
- `./gradlew :emulator:run`
- `./gradlew :emulator:jar`

## Entry point

- Main class: `server`

## Legacy artifacts

- Legacy `.bat` scripts and config/data files are preserved for compatibility.
- Gradle is now the preferred build path for cross-platform usage.
