## OpenRS2 317 Reference

This folder holds the official OpenRS2 archive reference client used for parity checks against the
native desktop renderer.

- Source: `https://archive.openrs2.org/clients/33303.dat`
- Artifact: `runescape-live-317-openrs2-33303.jar`
- Extracted class entry: `client.class`

The JAR is the obfuscated 317 client from `2005-06-13`.
